## âš¡ Ikhtisar Proyek

**NovaTrade Terminal** bukan sekadar aplikasi konversi mata uang biasa. Ini adalah *workstation* finansial tingkat profesional yang kami kemas dalam genggaman. Terinspirasi oleh estetika *Bloomberg Terminal*, aplikasi ini menggabungkan performa rendering tinggi Flutter dengan kecerdasan buatan dari Google Gemini.

Baik Anda sedang melacak volatilitas pasar, berkonsultasi dengan "Robovan" (AI kami) untuk saran finansial, atau menggunakan perintah suara untuk konversi mata uang secara *hands-free*, NovaTrade memberikan pengalaman premium yang tak tertandingi.

## ğŸ› ï¸ Teknologi & Stack

Aplikasi ini dibangun dengan standar industri tertinggi:

* **Framework:** `Flutter` (Dart) - Untuk performa *native* di semua platform.
* **State Management:** `Riverpod 2.0` - Manajemen state yang aman dan *testable*.
* **Artificial Intelligence:** `Google Gemini API` - Model generatif untuk analisis data.
* **Visualisasi Data:** `fl_chart` - Rendering grafik keuangan presisi tinggi dengan *custom painting*.
* **Voice Processing:** `speech_to_text` & `flutter_tts` - Integrasi suara dua arah.
* **Networking:** `Dio` - Koneksi API yang stabil dan cepat.
* **Animasi:** `flutter_animate` - Efek UI yang halus dan memukau.
