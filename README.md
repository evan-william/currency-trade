**NovaTrade Terminal** bukan sekadar aplikasi konversi mata uang biasa. Ini adalah **workspace finansial profesional** yang muat di saku Anda. Terinspirasi oleh estetika *Bloomberg Terminal*, aplikasi ini menggabungkan performa rendering tinggi Flutter dengan kecerdasan **Google Gemini AI**.

Apakah Anda ingin melacak volatilitas pasar, berkonsultasi dengan "Robovan" (AI Consultant), atau melakukan konversi mata uang hanya dengan perintah suara, NovaTrade menghadirkan pengalaman premium yang belum pernah ada sebelumnya.

## ğŸš€ Cara Menjalankan (Installation)

Ingin mencoba kecanggihan NovaTrade di mesin lokal Anda? Ikuti langkah ini:

1.  **Clone Repository**
    ```bash
    git clone [https://github.com/username-anda/NovaTrade-Terminal.git](https://github.com/username-anda/NovaTrade-Terminal.git)
    ```
2.  **Install Dependencies**
    Masuk ke folder project dan tarik semua library yang dibutuhkan.
    ```bash
    cd NovaTrade-Terminal
    flutter pub get
    ```
3.  **Konfigurasi API Key**
    * Dapatkan API Key gratis di [Google AI Studio](https://aistudio.google.com/).
    * Jalankan aplikasi, buka menu Chat, dan masukkan API Key Anda.
4.  **Run Application**
    ```bash
    flutter run
    ```
